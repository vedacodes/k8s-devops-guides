# k8s-devops-guides

A collection of concise, practical DevOps guides targeted at Kubernetes operators and platform engineers.

Contents:
- guides/01-intro.md - Kubernetes DevOps overview
- guides/02-gitops-argo.md - GitOps with Argo CD
- guides/03-ci-cd.md - CI/CD patterns for Kubernetes
- guides/04-helm.md - Packaging and managing apps with Helm
- guides/05-monitoring.md - Observability, Prometheus & Grafana
- guides/06-security.md - Kubernetes security best practices
- guides/07-backup-restore.md - Backup & restore strategies
- guides/08-networking.md - Networking & service mesh
- guides/09-troubleshooting.md - Troubleshooting checklist

Examples directory includes sample manifests and pipelines.
